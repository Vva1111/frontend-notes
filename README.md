- **CI/CD & Container:** Docker + GitLab CI/CD 파이프라인 구성, K8s 연동 
- **Cloud & Server:** Oracle Cloud 인스턴스 환경 조성, Nginx를 활용한 다중 프로젝트 배포 
- **Migration:** Node.js 버전 업그레이드 및 환경 세팅 -
