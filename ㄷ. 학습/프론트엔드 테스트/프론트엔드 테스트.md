
##### 단순히 UI 개체가 보이는지의 수준이 아니라, 렌더링 -> 이벤트 -> 상태 변화 -> DOM 반영 까지의 전반적인 흐름을 검증한다.

- 리팩토링 안전성 확보
- 회귀 버그 방지
- CI/CD 자동 검증
- 협업 시 신뢰성 확보
