- 파일을 생성할 수 있는 명령어

- 파일을 만들기 위해서는 명령어를 변수에 담아 사용해야 한다.
    
    ### EX )
    
    - abc = open( ‘파일 이름’, (열기 모드) ‘r’ / ‘a’ / ‘w’, encoding=’’ )
        
        → 열기 모드 : r - read( 읽기만 ) / a - append( 내용 추가) / w - write( 새로 쓰기 )
        
        → ==**`변수.write(’내용’)`**== 으로 입력
        
        → 입력 완료 후에는 항상 ==`**변수.close()**`==를 사용해줘야 한다
        
        ==`**with open(~~~~~~~~~~) as 변수**`== 형태로 사용하면 .close()를 사용할 필요가 없다.
        
        위의 형태를 사용 시 들여쓰기가 필요하고, 들여쓰기를 하지 않으면 자동으로 close됨
        
        → encoding은 생략 가능
