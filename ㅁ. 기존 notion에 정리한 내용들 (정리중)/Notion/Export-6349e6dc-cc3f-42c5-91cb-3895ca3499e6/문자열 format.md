- print( ) 를 사용해 문자를 출력하는 경우, ‘’ / “” 등이 중복돼 문자열 출력 오류 발생
    
    → 문자열 format 사용해 쉽게 문자열 출력
    
    1. print( ‘ 개발 언어에는 { a }, { b } 등이 있습니다. ’.format(a, b) )
    
    1. print( ‘ 개발 언어에는 { 1 }, { 0 } 등이 있습니다. ’.format(a, b) )
        
        ⇒ print( ‘ 개발 언어에는 { b }, { a } 등이 있습니다. ’.format(a, b) )
        
    
    1. print(f‘ 개발 언어에는 { a }, { b } 등이 있습니다. ’) ⇒ f-string
