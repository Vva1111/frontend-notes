- ==**`for in`**== : 정해진 범위 또는 데이터를 한번 회전
    
    - **for data in all_data** 의 형식으로 사용하며,
        
        → all_data 내부에 있는 데이터를 data 변수에 담아서 하나씩 반환
        
        → 예를 들어, all_data에 [ 1, 2, 3, 4, 50 ] 의 데이터가 있다면,
        
        for data in all_data:
        
        print(data)
        
        ⇒ 결과 값 : 1, 2, 3, 4, 50
        
    
    - **for data in range( int )** 의 형식으로도 사용할 수 있는데,
        
        → range(5) 는 0 이상 5 미만(4) 을 나타낸다
        
        → range(10) : 0 이상 10 미만 / range(1, 10) : 1 이상 10 미만
        
        / range(1, 10, 2) : 1부터 2씩 증가하며 9까지
        
        → range를 사용해 for in 반복문 범위를 직접 지정할 수 있다.
        
    

  

- ==`**while**`== : while문에 주어진 조건이 참(True)일 경우 반복문 내부 로직을 계속 실행한다.
    
    - while aaa<10:
        
        aaa += aaa ⇒ aaa 가 10 미만인 경우 계속 실행
        
        print(aaa) ⇒ aaa가 10이 됐을 때, 위의 로직을 실행하지 않고 빠져나와 실행
        
    

  

- ==**`break`**== : 반복문 내에서 어떠한 조건(주로 if문 사용해서)을 만족한다면 더 이상 반복하지 않고
    
    빠져나올 때 사용
    

- ==`**continue**`== : 반복문 내에서 어떠한 조건(if)을 만족하는 사항을 건너뛰고 반복문을 계속 실행
    
    ( → if문이 참인 경우에 건너뛴다 )
