⇒ 리스트 자료형 내에서 어떤 조건에 만족하는 값만 뽑아내거나, 값을 변형할 때 사용

  

### **Ex 1.)**

- 기존리스트 = [ ‘a’, ‘b’, ‘c’ ]

- 새로운리스트 = [ x + ‘추가함’ `for` x `in` 기존리스트 `if` ‘a’ `in` x ]
    
    → 기존 리스트 a, b, c를 for문으로 x에 가져온 뒤에 / 가져온 x 중 ‘a’ 가 포함돼 있는 값에 /
    
    ‘추가함’ 이라는 텍스트를 더한 값을 / 새로운 리스트에 넣는다.
    
    → 새로운리스트의 값은 [ ‘a 추가함’ ] 이 된다.
    
      
    

### **Ex 2.)**

- 기존리스트 = [ 1, 4, 7, 10, 11, 15, 19 ]

- 새로운리스트 = [str(int(x)-3) + ' 15보다는 큼' for x in basic_list if int(x) > 15 ]
    
    → 기존 리스트에서 반복문을 이용해 x에 값을 넘기게 되면 모두 문자열 데이터가 되므로,
    
    문자열 출력과 숫자 조건, 계산 시 타입을 명확하게 해줘야 한다.
    
    → 위의 결과로, 새로운리스트 = [ ‘15 15보다는 큼’, ‘19 15보다는 큼’ ] 이 반환된다
    
    ( x > 15인데 15가 출력 되는 이유 다시 알아보기 )
