## 토큰 목록 테이블 구현

- next.js 사용
    
    1. next.js 의 router 사용해서 직접 api 제공 ( /api/tokens )
    

  

### 필수 구현 기능

- [x] 최초 진입 시, /token-list 경로로 이동

- [x] GET /api/tokens API 사용해 /token-list 페이지에 토큰 목록 렌더링

- [x] 토큰 검색 기능 구현
    
    - 이름 또는 심볼에 검색어가 포함되어 있을 경우, 해당 항목만 표시되도록
    
    - 대소문자 구분하지 않음
    

- [x] 정렬 구현
    
    - market cap / volume / 24h% -> 예시 참조
    

- [x] API Response의 데이터 모델과 ui 데이터 모델이 같지 않은 상황으로 가정해서 작업

  

### 선택 구현 기능

- [x] 토큰 리스트 페이지에서 API로 토큰 리스트를 받아온 이후, 웹소켓으로 변경된 토큰 정보만 데이터가 내려온다고 요구사항이 변경되었을 때를 고려해 코드 작성 및 설명

- [x] 토큰 수가 수천 개 이상으로 증가할 경우, 현재 구조에서 발생할 수 있는 문제점과 그를 해결하기 위한 효율적인 데이터 처리 전략 고민

- [ ] 클라이언트 측의 렌더링 성능 외에도, 데이터 요청 및 관리 방식 차지에서 어느 전략이 효과적일지 함께 고민

- * 명시되지 않은 필요에 따른 컴포넌트 구조, 상태 관리 라이브러리의 사용 유무 및 선택은 자유
