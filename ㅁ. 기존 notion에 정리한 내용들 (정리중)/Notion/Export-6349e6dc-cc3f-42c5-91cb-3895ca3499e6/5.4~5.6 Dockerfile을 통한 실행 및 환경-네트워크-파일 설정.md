### **✅ 정리**

|명령어|설명|구문|예|
|---|---|---|---|
|RUN|컨테이너에서 명령 실행  <br>(주로 패키지 설치 등)|`RUN <실행하고 싶은 명령>`|`RUN apt-get update && apt-get install nginx`|
|CMD|컨테이너 실행 시 실행할 기본 명령|`CMD <실행하고 싶은 명령>`|`CMD ["nginx", "-g", "daemon off;"]`|
|ENTRYPOINT|필수 실행 명령 설정|`ENTRYPOINT <실행하고 싶은 명령>`|`ENTRYPOINT ["top"] CMD ["-d", "10"]`|
|ONBUILD|빌드 후 실행할 명령|`ONBUILD <실행하고 싶은 명령>`|`ONBUILD ADD website.tar /var/www/html/`|
|STOPSIGNAL|컨테이너 종료 시 송신할 시그널 지정|`STOPSIGNAL <시그널>`|`STOPSIGNAL SIGTERM`|
|HEALTHCHECK|상태 점검 명령|`HEALTHCHECK [옵션] CMD <실행할 명령>`|`HEALTHCHECK CMD curl -f <http://localhost/>`|
|ENV|환경 변수 설정|`ENV <key> <value>`  <br>`ENV <key>=<value>`|`ENV MY_NAME="Docker"`|
|WORKDIR|작업 디렉토리 설정|`WORKDIR <작업 디렉토리 경로>`|`WORKDIR /app`|
|USER|실행 사용자 지정|`USER <사용자명/UID>`|`USER myuser`|
|LABEL|이미지에 메타데이터 추가|`LABEL <키 명>=<값>`|`LABEL version="1.0”`|
|EXPOSE|컨테이너 포트 지정|`EXPOSE <포트 번호>`|`EXPOSE 8080`|
|ARG|빌드 시점에 사용할 변수 지정|`ARG <이름>[=기본값]`|`ARG VERSION=1.0`|
|SHELL|기본 쉘 지정|`SHELL ["쉘의 경로", "파라미터"]`|`SHELL ["/bin/bash", "-c"]`|
|ADD|파일 추가 및 압축 해제|`ADD <호스트의 파일 경로> <Docker 이미지의 파일 경로>`  <br>`ADD ["<호스트의 파일 경로>", "<Docker 이미지의 파일 경로>"]`|`ADD myfile.txt /app/`|
|COPY|단순 파일 복사|`COPY <호스트의 파일 경로> <Docker 이미지의 파일 경로>`  <br>`COPY ["<호스트의 파일 경로>", "<Docker 이미지의 파일 경로>"]`|`COPY myfile.txt /app/`|
|VOLUME|데이터 컨테이너와 호스트 간에 공유할 디렉토리 지정|`VOLUME <마운트 경로>`|`VOLUME ["/data"]`|
