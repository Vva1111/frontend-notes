- 여러 변수 / 함수를 묶어 한번에 관리할 수 있다.

- 클래스 내부에 각각의 객체가 있고, 이 객체들을 해당 클래스의 인스턴스라고 한다.

- 클래스 내부에 선언해 놓는 함수는 메소드라고 한다.

- 선언되는 클래스의 이름은 대문자로 시작하는 단어들의 조합으로 만든다
    
    - ex.) class ClassName
    

- a1 = ClassName 으로 a1 객체 인스턴스를 생성
    
    [a1.name](http://a1.name) = a1 인스턴스 내에서만 존재하는 name 변수 생성
    
    ⇒ a1 = ClassName class 속성 + name 변수
    

- def __init__(self, ~~) 함수를 사용해 멤버변수 생성 ( self는 기본이라고 생각 )

  

### self

- 메소드에서 인자 값을 전달 받을 때, 매개변수에 self 값을 사용하는데,
    
    이때 self는 객체 자기 자신을 지칭한다.
    

- 객체를 통해 메소드를 호출하는 경우,
    
    self 부분에 해당하는 전달 값은 따로 명시하지는 않는다
    
    하지만, 메소드가 받는 인자 값에 self는 항상 있어야 한다.
    

- 인스턴스가 속해있는 클래스가 가진 멤버변수 전체를 그대로 가져와 사용 가능하게 한다

- 다중 상속 시에는 사용하지 않는다

  

### 상속

```python
class ClassName(상속 받을 클래스 이름):
	def __init__(self, ~~~~~~):
		~~~~
```

- class ClassName(상속 받을 클래스):
    
    위의 형태로 다른 class를 상속 받을 수 있다. / 다중 상속도 가능
    

- 상속을 받게 되면 상속 받은 클래스의 메소드를 사용할 수 있다

- super ( 하위 클래스의 이름, 하위 클래스의 객체 ).함수
    
    - 상속의 대상인 상위 클래스를 호출할 때 사용하는 함수
    
    - 클래스 내부에서 super를 선언하면 인자 전달이 따로 필요하지 않다
    
    - 함수, 멤버 변수 모두 가져올 수 있다
    
    - `super().__init__( a, b )` 같은 경우에는
        
        현재 클래스에서 상위 클래스 __init__함수에 a, b 값을 전해줘
        
        함수 처리를 위해 사용한다.
